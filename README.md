**Mobile Installation Steps:**
===========================
1. Click file name
2. Click "View Raw" at bottom of page
3. Download the file to your phone
4. Open the APK file & Install

**Allow Restricted Permissions:**
1. Open the App info menu for PayPause. Either by holing on the app, or navigating from `settings -> apps -> manage apps`
2. Click the three dots in the top-right corner
   1. If you don't see this, click "Display Over Other Apps"
   2. Click on the grayed-out option
   3. Dismiss the "Access Denied" pop-up
   4. Now you can go back and click the three dots in the top-right corner
3. Click "Allow Restricted Settings"
4. Open the app & follow the in-app instructions!
